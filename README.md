# Course Registration System
A simple Course Registration System developed using Spring Boot, Thymeleaf, Spring Data JPA, and MySQL.
## Features
- View available courses
- Register for a course
- Store student details in MySQL
- View enrolled students
## Tech Stack
- Java 21
- Spring Boot
- Spring Data JPA
- Thymeleaf
- MySQL
- HTML
- CSS
